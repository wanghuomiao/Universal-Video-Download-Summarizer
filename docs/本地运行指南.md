## 第一步：克隆项目

git clone https://github.com/wanghuomiao/Universal-Video-Download-Summarizer.git
cd free-video-downloader


## 第二步：获取 API Key

项目需要 **1 个必需的 API Key**，另外 1 组为可选。



### 必需：DeepSeek API Key

DeepSeek 用于 AI 视频总结功能（字幕提取后调用大模型生成总结、思维导图、问答）。

1. 打开 [DeepSeek 开放平台](https://platform.deepseek.com/)，注册并登录
2. 进入「API Keys」页面，点击「创建 API Key」
3. 复制生成的 Key（以 `sk-` 开头），妥善保存


### 可选：Stripe 支付配置

Stripe 用于 VIP 会员的在线支付功能。可以不配置，不影响视频下载和 AI 总结等核心功能。

如果你想体验完整的支付流程：

1）访问 [Stripe Dashboard](https://dashboard.stripe.com/)，注册并登录

2）确保处于「测试模式」（Test mode）

3）在「开发者 → API 密钥」获取 Secret Key（以 `sk_test_` 开头）

4）在「产品」页面创建一个产品和价格，获取 Price ID（以 `price_` 开头）


5）安装 [Stripe CLI](https://docs.stripe.com/stripe-cli)，先执行 `stripe login` 登录：

然后运行以下命令启动本地 Webhook 转发：

stripe listen --forward-to localhost:8000/api/payment/webhook

终端会输出 Webhook Signing Secret（以 `whsec_` 开头）



## 第三步：配置环境变量

cd backend
cp .env.example .env

用任意文本编辑器打开 `backend/.env` 文件，填入你的配置：

env
# 必填 - DeepSeek API Key，用于 AI 视频总结
DEEPSEEK_API_KEY=sk-你的真实key

# 必填 - JWT 密钥，用于用户登录认证
# 本地开发可以随便写一个字符串，生产环境请使用强随机值
JWT_SECRET=my-local-dev-secret-123

# 选填 - Stripe 支付配置（不配置不影响核心功能）
STRIPE_SECRET_KEY=sk_test_你的stripe密钥
STRIPE_WEBHOOK_SECRET=whsec_你的webhook密钥
STRIPE_PRICE_ID_MONTHLY=price_你的价格id

# 选填 - 前端 URL，Stripe 支付完成后跳转地址
FRONTEND_URL=http://localhost:5173

 **注意**：`.env` 文件包含敏感信息，已被 `.gitignore` 排除，不会提交到代码仓库。


## 第四步：安装依赖

### 后端依赖（Python）

cd backend
pip install -r requirements.txt


### 前端依赖（Node.js）

cd ../frontend
npm install


## 第五步：初始化数据库

本项目使用 **SQLite**，**无需手动安装任何数据库软件**。

后端服务启动时会自动完成以下操作：
1. 在 `backend/data/` 目录下创建 `app.db` 数据库文件
2. 自动建表（`users` 用户表、`orders` 订单表）

💡 如果需要重置数据库，直接删除 `backend/data/app.db` 文件，重启后端即可自动重建。


## 第六步：启动项目

需要同时启动后端和前端，**打开两个终端窗口**：

**终端 1 启动后端：**

cd backend
python main.py

看到以下输出表示后端启动成功：

INFO:     Started server process [12345]
INFO:     Waiting for application startup.
INFO:     Application startup complete.
INFO:     Uvicorn running on http://0.0.0.0:8000 (Press CTRL+C to quit)

**终端 2 启动前端：**

cd frontend
npm run dev

看到以下输出表示前端启动成功：

VITE v7.x.x ready in xxx ms

➜  Local:   http://localhost:5173/



## 第七步：访问项目

打开浏览器，访问 http://localhost:5173 ，你将看到 AI 万能视频下载总结器的界面。



### 快速体验流程

1. 注册一个账号并登录
2. 在搜索框粘贴一个视频链接（如 B 站、YouTube、抖音），点击解析
3. 系统自动解析出视频标题、封面、可选格式，选择画质后下载
4. 解析视频后查看 AI 自动生成的总结和思维导图
5. 在 AI 总结页面输入问题，体验基于视频内容的智能问答

💡 AI 总结功能需要登录，免费用户每天可使用 3 次。
